# 2048 Clone (GitHub Pages demo)

A responsive 2048 clone built with plain HTML/CSS/JavaScript. Ready to deploy to GitHub Pages.

## Features

- Arrow keys + swipe controls
- Undo (single-step)
- Best score saved in `localStorage`
- Responsive layout and simple animations

## Files

- `index.html`
- `style.css`
- `script.js`

## Run locally

1. Clone the repo:
```bash
git clone https://github.com/pokasatyapriya/2048-clone.git
cd 2048-clone
```

2. Open `index.html` in your browser.
